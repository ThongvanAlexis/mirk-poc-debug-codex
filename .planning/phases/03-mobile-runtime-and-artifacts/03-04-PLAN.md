## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| GitHub Actions runner -> downloadable artifacts | CI output becomes the installable POC package. |
| macOS build job -> SideStore IPA | Unsigned iOS app bundle must be packaged in the exact IPA directory shape. |
| Workflow edits -> quality gates | Artifact additions must not bypass existing strict gates. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-3-10 | Tampering | CI artifact contents | mitigate | Package deterministic build output paths and test workflow commands/static paths. |
| T-3-11 | Denial of Service | iOS sideload pipeline | mitigate | Upload `.ipa` with `Payload/Runner.app`, not raw app bundle. |
| T-3-12 | Regression | Quality gates | mitigate | Keep artifact jobs dependent on gates and assert gate steps in workflow tests. |
</threat_model>
