<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Flutter asset bundle -> app support filesystem | Packaged bytes cross into mutable device storage. |
| Existing app support file -> trusted PMTiles path | A previous run may have left a partial or corrupt file. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-1-03 | Tampering | PMTiles copied file | mitigate | Validate byte length and SHA-256 before returning existing or newly written files. |
| T-1-04 | Denial of Service | PMTiles copy startup | mitigate | Surface copy errors in app state and tests; do not swallow failures or return invalid paths. |
| T-1-05 | Information Disclosure | File paths shown in app | accept | The app is a local POC and the copied app-support path is non-secret diagnostic information needed for MAP-02 evidence. |
</threat_model>
